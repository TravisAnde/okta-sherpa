Prerequisites

  • Access to all required machines (CL03, CL04, DC01, DC02)
  • Administrator credentials for Active Directory
  • Outbound internet connectivity on ports 80 and 443
  • Valid business email for Okta admin account
  • Modern web browser (Chrome or Firefox recommended)

Environment Setup

Machine Role Required Access Notes
CL03 Primary Setup Machine Internet Access Used for Okta Partner Portal and admin console access
